# Analisador de Desempenho Escolar 🎓

Este é um programa desenvolvido em **Java** para auxiliar no gerenciamento de notas acadêmicas. O sistema permite a entrada dinâmica de dados e identifica automaticamente o aluno com o melhor desempenho.

## 🚀 Funcionalidades
- Entrada personalizada da quantidade de alunos.
- Armazenamento de nomes e notas em arrays.
- Lógica de comparação para encontrar a maior nota da turma.
- Interface simples via terminal (CLI).

## 🛠️ Tecnologias Utilizadas
- **Java SE**
- **Scanner API** (para interação com o usuário)
